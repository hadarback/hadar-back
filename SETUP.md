# מערכת ניהול התורים — הדר בק | מדריך התקנה (v3)

מערכת שלמה שעובדת על **Google Sheet אחד + Apps Script + יומן Google** של הדר.
בלי שרת, בלי דאטה־בייס, בלי תשלום חודשי.

| קובץ | מה זה | דורש Deploy? |
|---|---|---|
| `index.html` | אתר התדמית | ❌ קובץ סטטי |

## מה המערכת עושה

- **קביעת תור** — בחירת טיפול → תאריך → שעה → פרטים. הטיפול אורך **שעה**, והמערכת שומרת **מרווח של 15 דק׳** בין מטופלות. לכן תור ב-12:00 חוסם אוטומטית את כל השעות עד **13:15** (60 דק׳ טיפול + 15 דק׳ מרווח). השעה הבאה שתוצע ללקוחה חדשה היא 13:15.
- **הסרת שעה כמשוריינת** — ברגע שמישהי קובעת, השעה (וכל חלון הטיפול+מרווח) נעלמת מהיומן הציבורי.
- **WhatsApp אוטומטי** — הודעת "הבקשה התקבלה" ללקוחה, הודעת אישור עם כתובת מלאה + "להגיע בבגדים נוחים", ותזכורת 24 שעות לפני.
- **יומן Google של הדר** — כשהדר מאשרת תור, נוצר אוטומטית אירוע ביומן שלה (עם שם, טלפון, כתובת).
- **"הוספה ליומן Google"** ללקוחה — בדף האישור.
- **מערכת ניהול** — אישור/דחיה/שינוי מועד, מאגר לקוחות, תיק מטופל, חסימת ימים, צפייה בהצהרת בריאות מלאה, תפוצת WhatsApp.

## התקנה (פעם אחת, ~20 דק׳)

### שלב 1 — Google Sheet + גיליונות
פתחי [sheets.google.com](https://sheets.google.com) → גיליון חדש בשם **"הזמנות הדר"**.
צרי 2 גיליונות (Tabs בתחתית, `+`). שאר הגיליונות ייווצרו לבד בפעם הראשונה.

**גיליון "הזמנות"** — שורה 1 (כל כותרת בתא נפרד):
```
id | תאריך_שליחה | שם | טלפון | שירות | משך_דקות | תאריך_פגישה | שעה_פגישה | הערות | סטטוס | health_id | calendar_event_id | תזכורת_נשלחה | אימייל
```

**גיליון "ימי_חופש"** — שורה 1:
```
תאריך | סיבה
```

> נוצרים אוטומטית בשימוש ראשון: **הצהרות_בריאות**, **לקוחות**, **תיק_מטופל**, **שעות_זמינות**.

> **מודל השעות:** ברגע שהדר פותחת שעות בטאב "שעות" (לוח חודשי + הוספה מהירה), דף ההזמנה
> מציג **רק** את השעות שהיא פתחה לאותו יום (פחות שעות תפוסות ומרווח). כל עוד לא נפתחו שעות
> בכלל — הדף נופל חזרה לשעות העבודה הקבועות, כך שלא נשבר כלום בתקופת המעבר.

### שלב 2 — אזור זמן (חשוב!)
**File → Settings → Time zone** של הגיליון = **(GMT+02:00) Jerusalem**.
זה קריטי כדי שאירועי היומן והתזכורות יהיו בשעה הנכונה.

### שלב 3 — Apps Script
1. בתוך הגיליון: **Extensions → Apps Script**.
2. מחקי את הקוד הקיים והדביקי את כל `apps-script.js`.
3. עדכני בראש הקובץ:
   ```js
   const ADMIN_PIN   = '1234';   // ← קוד גישה להדר
   const HADAR_EMAIL = '';       // ← מייל להתראות (אופציונלי)
   ```
4. **גם** ב-Apps Script: ⚙️ **Project Settings → Time zone → Asia/Jerusalem**.
5. שמרי (Ctrl+S).

### שלב 4 — Deploy
1. **Deploy → New deployment → Web app**.
2. **Execute as:** `Me` · **Who has access:** `Anyone`.
3. **Deploy** → אשרי הרשאות (`Advanced → Go to… (unsafe) → Allow`).
   ⚠️ ההרשאות כוללות **יומן Google** — זה תקין, צריך את זה כדי ליצור אירועים.
4. העתיקי את ה-**Web app URL** (מסתיים ב-`/exec`).

### שלב 5 — חיבור הקבצים
הדביקי את ה-URL ב-`const SHEETS_URL = '...'` ב-3 קבצים:
`booking.html` · `health.html` · `admin.html`.

### שלב 6 — תזכורות 24 שעות (טריגר)
ב-Apps Script: ⏰ **Triggers** (שעון בצד שמאל) → **Add Trigger**:
- Function: **`dailyReminders`**
- Event source: **Time-driven** → **Day timer** → **8am–9am**.
- שמרי. מעכשיו כל בוקר נשלחות תזכורות לכל התורים **המאושרים** של מחר.

## WhatsApp — שני מצבים

ברירת המחדל היא **`manual`** (עובד מיד, בלי הרשמה לשום שירות):

| | `manual` (ברירת מחדל) | `greenapi` (שליחה אמיתית אוטומטית) |
|---|---|---|
| אישור תור | נפתח אצל הדר WhatsApp עם הודעה מוכנה — לחיצה אחת | נשלח לבד ללקוחה |
| "הבקשה התקבלה" | מוצג ללקוחה במסך | נשלח לבד ללקוחה |
| תזכורת 24ש׳ | מייל להדר עם קישורי שליחה מוכנים | נשלח לבד ללקוחה |
| עלות / התקנה | אפס | חינמי בסיסי, דורש סריקת QR פעם אחת |

### להפעלת שליחה אוטומטית מלאה (Green API)
1. הרשמה ב-[green-api.com](https://green-api.com) → יצירת Instance → סריקת QR מהוואטסאפ של הדר.
2. ב-`apps-script.js`:
   ```js
   const WA_PROVIDER       = 'greenapi';
   const GREENAPI_INSTANCE = '1101000001';   // ה-idInstance שלך
   const GREENAPI_TOKEN    = '...';          // ה-apiTokenInstance שלך
   ```
3. Deploy גרסה חדשה (ראי למטה). מעכשיו כל ההודעות נשלחות אוטומטית.

## מערכת הניהול (admin.html)

- **הזמנות** — אישור / דחיה / בוצע · **שינוי מועד** (בלי ליצור הצהרה חדשה) · **בריאות** (צפייה בטופס המלא) · **תיק** (תיק מטופל) · **WA**.
- **מאגר לקוחות** — נבנה אוטומטית מכל מי שקובעת תור. אפשר להוסיף ידנית, לתייג, ולעשות **תפוצת WhatsApp**.
- **תיק מטופל** — לכל לקוחה: תאריך, סיבת פנייה, וסיכום טיפול שהדר כותבת.
- **ימים חסומים** — לסמן יום שלם כסגור (חופש/מחלה). היום נחסם לגמרי בקביעת תורים.

**סטטוסים:** `חדש` (ממתין) · `אושר` (חסום ביומן + נשלחה הודעה) · `נדחה` (משתחרר) · `בוצע` (היסטוריה).

⚠️ ה-PIN הוא ההגנה היחידה על האדמין. **לא לשתף את ה-URL של admin.**
