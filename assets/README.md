# 🖼️ תמונות — הדר בק

## תמונה ראשית של הדר

**קובץ:** `hadar.jpg`

כרגע יש כאן **placeholder אלגנטי** (גרדיאנט בצבעי המותג) — מתחזה לתמונה אמיתית כדי שהדף יעבוד.

### החלפה לתמונה האמיתית

1. בחרי תמונה של הדר (פורטרט/חצי גוף, רצוי על רקע נקי)
2. **חתכי** ל-יחס 3:4 (אנכי, ~900×1200px לפחות)
3. שמרי כ-**JPG** עם השם המדויק `hadar.jpg`
4. החליפי את הקובץ הזה (`assets/hadar.jpg`)
5. ה-cache של GitHub Pages לפעמים תופס לכמה דקות — אם זה לא מתעדכן מיד, רענני עם Ctrl+Shift+R

### הנחיות לתמונה אידאלית

✔ פורטרט/חצי גוף, אנכית
✔ אור טבעי רך, לא חזק
✔ רקע נקי או מטושטש
✔ מבט רגוע
✔ צבעים חמימים (יתאימו לפלטה: קרם / טראקוטה / כחול-אבק)

✘ לא סלפי באור ניאון
✘ לא תמונות קבוצתיות
✘ לא רקע צבעוני/עמוס
